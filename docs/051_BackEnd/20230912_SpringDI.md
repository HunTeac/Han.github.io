# BackEnd

## Spring Framework

### Spring Framework 특징
- POJO(Plain Old Java Object) 방식의 프레임워크
- 의존성 주입(Dependency Injection)을 통한 객체관계 구성
- 관점지향 프로그래밍(AOP, Aspect Oriented Programming)
- 제어 역전(IoC, Inversion of Control)
- 높은 확장성과 다양한 라이브러리
